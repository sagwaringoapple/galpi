# 갈피 (Galpi)

장기 롤플레이의 전개를 AI와 상담하고, 아이디어를 스토리보드 카드로 관리하며,
선택한 카드만 프롬프트에 반영하기 위한 SillyTavern 확장입니다.

## v0.1.1

현재 구현된 기능:

- 갈피 패널
- 최근 롤플 메시지 불러오기
- 상담용 프롬프트 생성
- AI 추천 결과를 스토리보드 카드로 저장
- 카드 생성·수정·삭제
- 이번 응답 / 지속 주입 선택
- 실제 주입 프롬프트 미리보기 및 복사

현재 v0.1.1에서는 AI 상담 자동 실행과 실제 생성 프롬프트 자동 삽입은 아직 연결되지 않았습니다.
우선 SillyTavern에서 확장이 정상 설치·로드되고, 카드 저장과 미리보기가 작동하는지 확인하는 단계입니다.
