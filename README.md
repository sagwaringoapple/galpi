# 갈피 (Galpi)

장기 롤플레이를 위한 SillyTavern 스토리 보조 확장입니다.

## v0.1.0

이번 버전은 실제 SillyTavern에서 검증된 메뉴/모달 열기 방식을 사용한 구조 테스트 버전입니다.

- 확장 설정 탭에 전역 설정 표시
- 채팅 화면 확장 메뉴에 갈피 표시
- 메뉴 팝업을 먼저 닫은 뒤 80ms 후 갈피 창 열기
- 모바일 visualViewport 기준 위치 보정
- 스토리 카드 / AI 상담 / 주입 탭의 기본 UI

스토리 카드 저장, AI 상담, 실제 프롬프트 주입은 다음 버전에서 연결합니다.
