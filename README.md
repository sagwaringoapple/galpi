# 갈피 (Galpi)

장기 롤플레이를 위한 SillyTavern 스토리 보조 확장입니다.

## v0.0.2 — 구조 테스트

- 확장 설정 탭에 전역 설정 표시
- 채팅 화면의 확장 메뉴에 `갈피` 항목 표시
- 별도 모달 창 열기
- 현재 캐릭터명 표시
- 채팅방별 작업 메모 저장
- 채팅방별 스토리 카드 생성·수정·삭제
- 모바일 전체 화면 레이아웃
- 자동 저장 및 수동 저장

AI 상담, 실제 프롬프트 주입, 프롬프트 미리보기는 이후 버전에서 연결합니다.
